== Description ==
Foodmania theme is specially designed for users having food websites however can be used for mulitple purposes.
The theme is genetically responsive to provide an amazing experience for iPhone, iPad &amp; Android users.
Foodmania theme is super flexible and has a fully responsive design. Built with HTML5, CSS3, Foundation 5, Grunt.
Foodmania comes with built in support for multiple plugins like rtMedia, BuddyPress, bbPress, WooCommerce, Gravity Forms and much more.

== Plugins Supported ==

* bbPress - http://wordpress.org/plugins/bbpress/

* BuddyPress - http://wordpress.org/plugins/buddypress/

* BuddyPress Docs - http://wordpress.org/plugins/buddypress-docs/

* Jetpack - https://wordpress.org/plugins/jetpack/

* Ninja Forms - http://wordpress.org/plugins/ninja-forms/

* Regenerate Thumbnails - http://wordpress.org/plugins/regenerate-thumbnails/

* rtSocial - https://wordpress.org/plugins/rtsocial/

* rtWidgets - https://wordpress.org/plugins/rtwidgets/

* rtMedia - http://wordpress.org/plugins/buddypress-media/

* rtMedia-Pro - https://rtcamp.com/rtmedia/addons/pro/

* WooCommerce - http://wordpress.org/plugins/woocommerce/

* WordPress SEO by Yoast - http://wordpress.org/plugins/wordpress-seo/

* Yet Another Related Posts Plugin - https://wordpress.org/plugins/yet-another-related-posts-plugin/

== Important Note ==
If you make changes to thumbnail height, width or crop settings, you must use "Regenerate Thumbnail Plugin" ( http://wordpress.org/extend/plugins/regenerate-thumbnails/ ) to regenerate thumbnails on old posts.
